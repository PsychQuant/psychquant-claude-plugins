# 公理J1: 間 (Ma) - 余白の美学

## 定義

- 物語は表現されたものとあえて表現されなかったものの両方から生まれる
- 言葉にされないことが、読者の参加と推論のための認知的空間を作り出す
- 時間的・感情的な間（ポーズ）は物語の本質的な構造要素である

## 形式的表現

任意の物語要素eについて、その意味M(e)はその内容C(e)と周囲の余白N(e)の関数である

```
M(e) = f(C(e), N(e))
```

## 美学的背景

- 日本美学における「余白」の概念
- 水墨画、書道、庭園における空間の活用
- 禅の「不立文字」思想
